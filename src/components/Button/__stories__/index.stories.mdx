# Button

<Description>
  Button component is commonly used for actions into a web application. You can
  set various of settings to let it have the behavior you want.
</Description>

## Basic

<Canvas>
  <Story name="Basic">
    <Button>BUTTON</Button>
  </Story>
</Canvas>

## Variants

<Description>Set variant using `variant` property.</Description>

<Canvas>
  <Story name="Variants">
    {buttonVariants.map(variant => (
      <Button key={variant} variant={variant} m={2}>
        {variant}
      </Button>
    ))}
  </Story>
</Canvas>

## Sizes

<Description>Set size using `size` property.</Description>

<Canvas>
  <Story name="Sizes">
    {buttonSizes.map(size => (
      <Button key={size} size={size} m={2}>
        {size}
      </Button>
    ))}
  </Story>
</Canvas>

## Disabled

<Description>Set disabled using `disabled` property.</Description>

<Canvas>
  <Story name="Disabled">
    <Button disabled m={2}>
      disabled
    </Button>
  </Story>
</Canvas>

## Progress

<Description>
  Add `ActivityIndicator` by setting `progress` property to `left` or `right`.
</Description>

<Canvas>
  <Story name="Progress">
    <Button progress="left" m={2}>
      left progress
    </Button>
    <Button progress="right" m={2}>
      right progress
    </Button>
  </Story>
</Canvas>

## Progress Only

<Description>
  Add `displayProgressOnly` if you want your button to have only a progress
  circle in it.
</Description>

<Canvas>
  <Story name="Progress Only">
    <Button displayProgressOnly progress="right" m={2} />
  </Story>
</Canvas>

## Icons

<Description>
  Add icon using `icon`. You can specify the name of the icon or the icon
  itself.
</Description>

<Canvas>
  <Story name="Icons">
    <Button icon="lock" m={2} />
    <Button icon="lock" m={2}>
      With text
    </Button>
  </Story>
</Canvas>

## Icons Positions

<Description>
  Add a position icon using `iconPosition`. You can specify whether you want it
  to the left or to the right of your text. It requires an `icon`.
</Description>

<Canvas>
  <Story name="Icons Position">
    {['left', 'right'].map(position => (
      <Button key={position} iconPosition={position} icon="lock" m={2}>
        {position}
      </Button>
    ))}
  </Story>
</Canvas>

## Extend

<Description>
  Make button extensible using `extend` prop. It requires a `icon`.
</Description>

<Canvas>
  <Story name="Extend">
    <Button extend icon="plus" m={2}>
      extend
    </Button>
  </Story>
</Canvas>

## Download

<Description>
  Use `download` prop if you want to make it a downloadable button.
</Description>

<Canvas>
  <Story name="Download">
    <Button download m={2} icon="download" />
  </Story>
</Canvas>

## Link delegation

<Description>
  Using the `to` props, link behavior is delegated to [Link
  Component](docs-components-link).
</Description>

<Canvas>
  <Story name="Link">
    <Button to="https://scaleway.com" target="_blank">
      Scaleway
    </Button>
  </Story>
</Canvas>
