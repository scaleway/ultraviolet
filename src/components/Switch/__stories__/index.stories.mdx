# Switch

## Basic

<Description>
  `Switch` uses a `input` of type &quot;checkbox&quot; and they can only be used
  as controlled input.
</Description>

<Canvas>
  <Story name="basic">
    <ControlValue value={false}>
      {({ value, onChange }) => (
        <Switch
          checked={value}
          name="switch-basic"
          onChange={e => {
            onChange(e.target.checked)
          }}
        />
      )}
    </ControlValue>
  </Story>
</Canvas>

## Labeled

By setting `labeled` prop you can put the label in the `Switch` component

<Canvas>
  <Story name="labeled-basic">
    <ControlValue value={false}>
      {({ value, onChange }) => (
        <Switch
          name="switch-label"
          labeled
          variant="primary"
          checked={value}
          onChange={e => onChange(e.target.checked)}
        />
      )}
    </ControlValue>
  </Story>
</Canvas>

### Custom on/off texts

Default label placement is inside the Switch. you can customize the text with `onLabel` and `offLabel`props.

<Canvas>
  <Story name="label-inside">
    <ControlValue value>
      {({ value, onChange }) => (
        <Switch
          name="switch-label-custom"
          labeled
          onLabel="YES"
          offLabel="NO"
          variant="primary"
          checked={value}
          onChange={e => onChange(e.target.checked)}
        />
      )}
    </ControlValue>
  </Story>
</Canvas>

### Label placement

By setting the `labelPlacement` prop you can choose where you want to put the label. Moreover you can customise the label rendered with the `labelRender` prop.

<Canvas>
  <Story name="label-placement">
    <BlockList>
      <ControlValue value>
        {({ value, onChange }) => (
          <Switch
            name="switch-label-right"
            labelPlacement="right"
            labeled
            variant="primary"
            checked={value}
            onChange={e => onChange(e.target.checked)}
          />
        )}
      </ControlValue>
      <ControlValue value>
        {({ value, onChange }) => (
          <Switch
            name="switch-label-left"
            labelPlacement="left"
            labeled
            onLabel="CHECKED"
            offLabel="NOT CHECKED"
            variant="primary"
            checked={value}
            onChange={e => onChange(e.target.checked)}
          />
        )}
      </ControlValue>
    </BlockList>
  </Story>
</Canvas>

### Custom Label render

If you choose to place label near the Switch component with `labelPlacement` you can set a custom element for the label with the `labelRender` prop.

<Canvas>
  <Story name="custom-label-render">
    <ControlValue value>
      {({ value, onChange }) => (
        <Switch
          name="switch-label-left"
          labelPlacement="right"
          labeled
          labelRender={
            <span
              style={{
                marginLeft: 10,
                color: value ? '#4F0599' : 'inherit',
                fontWeight: value ? 'bold' : 'normal',
              }}
            >
              Custom label render
            </span>
          }
          variant="primary"
          checked={value}
          onChange={e => onChange(e.target.checked)}
        />
      )}
    </ControlValue>
  </Story>
</Canvas>
