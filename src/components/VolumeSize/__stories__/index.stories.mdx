# VolumeSize

A beautiful component to show a bar with limits

<Canvas>
  <Story name="Basic">
    <VolumeSize
      maxLabel="max"
      maxSize={40}
      minLabel="min"
      minSize={0}
      requiredLabel="required"
      title="Basic"
      unit="GB"
      value={0}
    />
  </Story>
</Canvas>

## Min and Max size

You can set max and min size with `maxSize` and `minSize`.

<Canvas>
  <Story name="Min and Max size">
    <VolumeSize
      maxSize={70}
      minSize={30}
      title="Min and Max size"
      unit="KB"
      value={40}
    />
  </Story>
</Canvas>
