Installation:
```bash
cd e2e
pnpm add -D @axe-core/playwright axe-core
```

Helper à créer (e2e/helpers/a11y.ts):
```typescript
import { AxeBuilder } from '@axe-core/playwright'
import type { Page } from '@playwright/test'

export const testA11y = async (page: Page, path: string) => {
  await page.goto(path)
  
  const accessibilityScanResults = await new AxeBuilder({ page })
    .withTags(['wcag2a', 'wcag2aa'])
    .analyze()
  
  expect(accessibilityScanResults.violations).toEqual([])
}
```

Test example (e2e/tests/button-a11y/test.spec.ts):
```typescript
import { test } from '@playwright/test'
import { testA11y } from '../../helpers/a11y'

test('Button - A11y', async ({ page }) => {
  await testA11y(page, '/button')
})
```

Règles e2e spécifiques:
- landmark-one-main (activé en e2e, pas en unit)
- region (activé en e2e)
- page-has-heading-one (activé)
- skip-link (activé)
