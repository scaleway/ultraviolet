Documentation à créer:
```markdown
# Screen Reader Testing Guide

## Quick Start
- NVDA: Ctrl+Alt+N
- VoiceOver: Cmd+F5
- JAWS: Insert+F5

## Key Commands
### NVDA
- Navigate: Tab, Arrow keys
- Read all: CapsLock+Down
- Stop speech: Ctrl

### VoiceOver
- Navigate: VO+Arrow keys
- Read all: VO+A
- Stop speech: Ctrl

## Testing Checklist
[ ] Page title announced
[ ] Main landmarks announced
[ ] Form labels announced
[ ] Error messages announced
[ ] Dynamic changes announced
[ ] Focus order logical
```
