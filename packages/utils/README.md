# @repo/utils

Shared utilities for the repository.

## Usage

```javascript
import { capitalize, getUUID, orderBy } from '@ultraviolet/utils'
```

## Utilities

- `capitalize` - Capitalizes the first letter of a string
- `getUUID` - Generates a UUID string
- `orderBy` - Creates a comparator function for sorting arrays
- ... (more utilities will be added here)
