## Best Practices

### ✅ Do

```tsx
<Row templateColumns="1fr 1fr" gap="medium">
  <article>
    <h2>Card Title</h2>
    <p>Content</p>
  </article>
  <article>
    <h2>Card Title</h2>
    <p>Content</p>
  </article>
</Row>

<Row templateColumns={{ xxsmall: '1fr', medium: '1fr 1fr' }}>
  <FormSection>Personal Info</FormSection>
  <FormSection>Address</FormSection>
</Row>
```

### ❌ Avoid

```tsx
// Do not use for tabular data
<Row templateColumns="1fr 1fr">
  <div>Name</div>
  <div>Age</div>
</Row>
// → Use the dedicated Table component

// Do not break reading order
<Row templateColumns="1fr 1fr">
  <div>First</div>
  <div aria-hidden>Hidden</div>
  <div>Second (visuellement décalé)</div>
</Row>
```
