# Row - Accessibilité

## Résumé

**Statut** : ✅ Compliant  
**Type** : Layout Primitive (non-interactif, CSS Grid)

## Checks rapides

- ✅ Clavier : N/A (non-interactif)
- ✅ ARIA : Utilise `div` natif
- ✅ Focus : Ordre DOM = ordre visuel
- ✅ Contraste : N/A (transparent)
- ✅ Screen Readers : Transparent (approprié)

## Actions requises

Aucune - composant déjà compliant.

## Bonnes pratiques

### ✅ Faire

```tsx
<Row templateColumns="1fr 1fr" gap="medium">
  <article>
    <h2>Card Title</h2>
    <p>Content</p>
  </article>
  <article>
    <h2>Card Title</h2>
    <p>Content</p>
  </article>
</Row>

<Row templateColumns={{ xxsmall: '1fr', medium: '1fr 1fr' }}>
  <FormSection>Personal Info</FormSection>
  <FormSection>Address</FormSection>
</Row>
```

### ❌ Éviter

```tsx
// Ne pas utiliser pour des données tabulaires
<Row templateColumns="1fr 1fr">
  <div>Name</div>
  <div>Age</div>
</Row>
// → Utiliser le composant Table dédié

// Ne pas casser l'ordre de lecture
<Row templateColumns="1fr 1fr">
  <div>First</div>
  <div aria-hidden>Hidden</div>
  <div>Second (visuellement décalé)</div>
</Row>
```
