## Best Practices

### ✅ Do

```tsx
<Stack gap="medium">
  <Label>Input Label</Label>
  <Input />
  <HelperText>Helper message</HelperText>
</Stack>

<Stack as="section">
  <h2>Section Title</h2>
  <p>Content</p>
</Stack>
```

### ❌ Avoid

```tsx
// Do not use for semantic lists
<Stack role="list">
  <div>Item 1</div>
  <div>Item 2</div>
</Stack>
// → Use the dedicated List component

// Do not hide interactive content
<Stack aria-hidden>
  <Button>Action</Button>
</Stack>
```
