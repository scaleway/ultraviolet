## Bonnes pratiques

### ✅ Faire

```tsx
<Stack gap="medium">
  <Label>Input Label</Label>
  <Input />
  <HelperText>Helper message</HelperText>
</Stack>

<Stack as="section">
  <h2>Section Title</h2>
  <p>Content</p>
</Stack>
```

### ❌ Éviter

```tsx
// Ne pas utiliser pour des listes sémantiques
<Stack role="list">
  <div>Item 1</div>
  <div>Item 2</div>
</Stack>
// → Utiliser le composant List dédié

// Ne pas cacher du contenu interactif
<Stack aria-hidden>
  <Button>Action</Button>
</Stack>
```
